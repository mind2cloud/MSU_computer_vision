Обнаружить отрезки прямых на контурном изображении, полученном с помощью оператора Кэнни (из задания 1).
Построить результаты в виде наложения обнаруженных отрезков на исходном кадре видеопоследовательности.
(Для каждого 10-го кадра, но этот параметр сделать регулируемым).
Оценить количество отрезков, обнаруживаемых при разных значениях параметров точности алгоритма обнаружения.

1) Преобразование в черно-белое, сглаживание 3х3, контуры - как в Задании 1.
2) Применить функцию OpenCV для обнаружения отрезков прямых с помощью преобразования Хафа (HoughLinesP).
   Параметры преобразования - точность обнаружения прямых положению (rho) и по углу (theta).
   Использовать три набора значений параметров (rho, theta): (10 /*пикс*/, 15 /*град*/), (5, 5), (1, 1).
3) Для каждого набора параметров записать выходные изображения (цветные кадры с наложением прямых) в 
   отдельные папки.
4) Сформировать журнальный текстовый файл. Этот текстовый файл в каждой строке
   должен содержать: номер кадра, количество обнаруженных отрезков для первой пары параметров, для второй и для третьей.
   Например:   100  500  1500  5000
